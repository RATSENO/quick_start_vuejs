# contactsapp_search  
```  
vue init webpack-simple contactsapp_search  
npm install  
npm install --save axios vuex es6-promise

```

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
